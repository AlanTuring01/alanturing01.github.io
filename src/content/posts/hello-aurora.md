这个博客从 2022 年的一个 Bootstrap 单页模板，长成了现在的样子。新版的设计主题叫**星港极光**。

我在海事大学读人工智能——深空与深海、信号灯与航标，这些意象就是从这里来的。页面上那条青色到品红的渐变是全站唯一的"极光"，分割线里藏着莫尔斯电码式的短划，文章读完时会点亮一枚"读讫"信号灯。

## 技术上的变化

- 从 jQuery + Bootstrap 单页，重构为 [Astro](https://astro.build) 静态多页站，文章用 Markdown 写作；
- 零外部依赖：不加载任何 CDN、字体或第三方脚本（留言板的评论组件需要你点一下才会加载）；
- 全站深浅两套主题，跟随系统偏好，也可以手动切换；
- 相册与视频做了大幅压缩，加载速度快了一个数量级；
- 部署由 GitHub Actions 自动完成，源码依旧[完全开源](https://github.com/AlanTuring01/alanturing01.github.io)。

老版本的一切都还在 git 历史里，包括那首一直在页角转动的 Flower Dance——它现在住在右下角的小唱片里。

欢迎在[留言板](/guestbook/)坐坐。
